# pk_aava_code_exam

## Usage instructions:

## Step 1: get this repo

### git clone https://github.com/ppkettu/pk_aava_code_exam.git

## Step 2a: Run the static version

### cd pk_aava_code_exam.git
### serve -s build

(ctrl+C when done)

## Step 2b: Development build

### cd pk_aava_code_exam.git
### npm i
### npm start
